Based on the Voibos web services, the walking time calculation for the selected or all features is returned, and the tables can then be updated. More information about the Voibos walking time calculation application: https://www.geoland.at/site/geoservices.html.

To use the plugin, select the feature or features you want to calculate the walking times for, and click on the "calculate" button.

To calculate all the layer features, tick the first checkbox and click on the "calculate all" button.

To update the attribute values of your layer with the calculation values, tick the second checkbox and select on the table in which layer attribute you want to store each calculated value.
